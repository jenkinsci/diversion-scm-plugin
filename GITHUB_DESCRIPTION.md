## How to Set Description

1. Go to your repository: https://github.com/ibain/jenkins-diversion-scm-plugin
2. Click the ⚙️ (Settings) icon next to "About"
3. Paste the description in the "Description" field
4. Add topics in the "Topics" field
5. Click "Save changes"
