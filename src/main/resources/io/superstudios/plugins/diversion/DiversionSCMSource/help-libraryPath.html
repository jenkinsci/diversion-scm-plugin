The base path within the repository where the library files are located.

Expected Structure:

  • vars/ - Global variables and functions (e.g., vars/uploadToGooglePlay.groovy)
  • src/ - Shared classes (optional)
  • resources/ - Resource files (optional)

Examples:

  • Meta/Jenkins/SharedLib - If your library is at Meta/Jenkins/SharedLib/vars/
  • jenkins-lib - If your library is at repository root jenkins-lib/vars/
  • . or leave empty - If vars/ is at repository root

Jenkins will look for vars/, src/, and resources/ directories within this base path.
